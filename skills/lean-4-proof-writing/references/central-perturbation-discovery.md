# Central Perturbation Discovery (2026-06-08)

## Summary

Constructed an (n-1)-dimensional family of 1/2-derivations on N_n (strictly upper
triangular n×n matrices) that are NOT scalar multiples of the identity. This
contradicts the claimed theorem `Der_{1/2}(N_n) ≅ F` (n≥5) — our construction
shows `dim ≥ n`.

## The construction

For each adjacent source `E(i,i+1)` (i = 1, …, n-2), define:

```
C_i(E(i,i+1)) = E(1,n)
C_i(E(p,q))   = 0     for all other basis elements
```

**Lean verification:** `centralPert` in session 2026-06-08 compiles
as a valid `HalfDerivation F n` via the standard `half_deriv_cond`:

```lean4
noncomputable def centralPert (lam : ℕ → F) : HalfDerivation F n where
  coeff x y :=
    if hx_adj : x.j = x.i + 1 then
      if hy_corner : y.i = 1 ∧ y.j = n then
        lam x.i
      else
        0
    else
      0
  half_deriv_cond := by
    intro x y k hk hk'
    by_cases hx_adj : x.j = x.i + 1
    · rw [hx_adj] at hk'; omega
    · have hk_lt_n : k < n := lt_of_lt_of_le hk' x.hle
      have hx_i_lt_n : x.i < n := lt_of_lt_of_le x.hlt x.hle
      have hx_j_ne_one : x.j ≠ 1 := by omega
      have hx_i_ne_n : x.i ≠ n := by omega
      have hk_ne_one : k ≠ 1 := by omega
      simp [hx_adj]
      simp [show k ≠ n from by omega, hx_j_ne_one, hx_i_ne_n, hk_ne_one]
```

### Verification: E(1,n) is central

[E(a,b), E(1,n)] = δ_{b,1}·E(a,n) - δ_{n,a}·E(1,b)

Since a < b ≤ n, we have b ≠ 1 and a ≠ n, so both δ terms vanish → bracket = 0.
Verified in Lean for n=5.

### Why established properties hold

| Property | Reason |
|----------|--------|
| **Half-derivation condition** | Adjacent source: no k exists (vacuous). Nonadjacent: all coeffs = 0, and RHS targets can't be (1,n) |
| **Interior Type C (v < n)** | `cond_zero(i,i+1,v,v+1,u,v+1)` — covered by separate lemma |
| **Type A/B propagation** | Right/left bracket equations only constrain coefficients at targets (u,i+1), (i,v) — NOT (1,n) |
| **cond_zero coverage** | Exhaustive term classification shows `f i(i+1) 1 n` doesn't appear in ANY cond_zero equation |

## Structural interpretation

The discovery fits a general theory: for a nilpotent Lie algebra L,

```
Der_{1/2}(L) ≅ F·Id ⊕ Hom(L/[L,L], Z(L))
```

For N_n (strictly upper triangular n×n matrices):
- L/[L,L] ≅ F^{n-1} (generated by adjacent sources E(i,i+1))
- Z(L) ≅ F (generated by E(1,n), the longest root)
- Hom(L/[L,L], Z(L)) ≅ F^{n-1}
- Total: dim = 1 + (n-1) = n

Each C_i maps a single adjacent generator to E(1,n) and everything else to 0.

## half_deriv_cond audit: the 4-term formula vs standard definition

Session 2026-06-08 performed a line-by-line audit of the `half_deriv_cond`
field against the standard half-derivation definition:

**Standard:** `2·D([x,y]) = [D(x), y] + [x, D(y)]` for all x, y

**Lean form** (for x = E(i,j), y = E(u,v), any k with i < k < j):
```
2·coeff(i,j,u,v) = 
  (if v=j ∧ u<k then coeff(i,k,u,k) else 0)    -- IA: [E(p,k), E(k,j)] = E(p,j)
- (if u=k ∧ j<v then coeff(i,k,j,v) else 0)    -- IB: [E(j,q), E(k,j)] = -E(k,q)  
- (if v=k ∧ u<i then coeff(k,j,u,i) else 0)    -- IIA: [E(i,k), E(p,i)] = -E(p,k)
+ (if u=i ∧ k<v then coeff(k,j,k,v) else 0)    -- IIB: [E(i,k), E(k,q)] = E(i,q)
```

**Derivation:** Apply the standard condition to x₁=E(i,k), x₂=E(k,j) where
[x₁,x₂] = E(i,j). Expand both sides and take coefficient of E(u,v).

Each of the 4 if-terms corresponds to one of the 4 non-zero bracket cases
in `[D(x₁), x₂] + [x₁, D(x₂)]`:

| Term | Source | Target | Bracket condition |
|------|--------|--------|-------------------|
| T1 (IA) | D(E(i,k)) | E(u,k) | [E(u,k), E(k,j)] = E(u,j), requires v=j |
| T2 (IB) | D(E(i,k)) | E(j,v) | -[E(j,v), E(k,j)] = -E(k,v), requires u=k |
| T3 (IIA) | D(E(k,j)) | E(u,i) | -[E(i,k), E(u,i)] = -E(u,k), requires v=k |
| T4 (IIB) | D(E(k,j)) | E(k,v) | [E(i,k), E(k,v)] = E(i,v), requires u=i |

**Conclusion:** The 4-term formula IS exactly the standard definition,
expanded in coefficient form. No encoding errors.

## Paper gap analysis

The Lean code comments in `Diagonalization.lean` state:

> "ADJACENT SOURCES (j = i+1): NOT covered by this lemma. The paper's
> Theorem 4.2 has the same gap — there is no k with i<k<j"
> "This is a genuine gap in the paper's Theorem 4.2."

The code authors' judgment was that this gap does not affect the main
classification (n ≥ 5, D = c·Id), because the RecursiveSystem only uses
diagonal coefficients C(i,j) for j-i ≥ 2.

**Our finding: This judgment was incorrect.** The central perturbation is a
counterexample that satisfies ALL constraints the paper's proof imposes
(nonadjacent off-diagonal = 0, all diagonal coefficients equal), yet is
NOT c·Id. The paper's unstated assumption was that nonadjacent off-diagonal
zero + all-diagonal-equal implies D = c·Id, which the central perturbation
falsifies.

### The reasoning chain

```
Paper's Theorem 4.2: nonadjacent off-diagonal = 0  (has adjacent gap, noted in code)
    ↓
RecursiveSystem: all diagonal C(i,j) equal for n≥5  (correctly proven in Recursion.lean)
    ↓
Paper claims: D = c·Id                                ← WRONG
```

The gap: the paper's third step assumes the only remaining unknowns are
diagonal coefficients. The central perturbation has NO nonadjacent
off-diagonals and ALL zero diagonals, yet is NOT zero.

## Structural explanation: Hom(L/[L,L], Z(L))

The family we discovered has a unified algebraic explanation:

```
Der_{1/2}(L) ≅ F·Id ⊕ Hom(L/[L,L], Z(L))
```

For N_n (strictly upper triangular n×n matrices):
- L/[L,L] ≅ F^{n-1} (generated by adjacent sources E(i,i+1))
- Z(L) ≅ F (generated by the longest root E(1,n))
- Horn(L/[L,L], Z(L)) ≅ Hom(F^{n-1}, F) ≅ F^{n-1}
- Total dimension = 1 + (n-1) = n

**Why this works:** E(1,n) is central in N_n ([E(a,b), E(1,n)] = 0 for all
basis elements). Adjacent sources E(i,i+1) are not in [L,L] (they generate
L modulo [L,L]). The half-derivation condition 2·D([x,y]) = [D(x),y] + [x,D(y)]
is trivially satisfied because:
- [C(x), y] = 0 always (C(x) is a multiple of central E(1,n))
- [x, C(y)] = 0 always (same reason)
- C([x,y]) = 0 always (because [L,L] has no adjacent sources)

This gives n independent directions: Id + n-1 central perturbations.

## Key Lean techniques used

### Constructing a perturbation and verifying at structure level

Instead of working with NCoeff (which has a bounded axiom set that can't express
the corner coefficient), work directly with the `HalfDerivation` structure.
The `half_deriv_cond` field quantifies over all k, and adjacent sources have
empty k-range (vacuous truth), making the verification straightforward.

### Proving `(if cond then a else 0) = 0` when `¬cond`

`simp [h]` doesn't handle `h : ¬P`. Use `ite_eq_right_iff.mpr`:

```lean4
have h_term : (if P then D.f a b c d else 0) = 0 := by
  apply ite_eq_right_iff.mpr
  intro h
  exfalso; exact hP_not h    -- where hP_not : ¬P
```

### The `half_deriv_cond` derivation audit

The 4-term formula in `HalfDerivation.half_deriv_cond` is derived from:

```
2·D([E(i,k), E(k,j)]) = [D(E(i,k)), E(k,j)] + [E(i,k), D(E(k,j))]
```

Since [E(i,k), E(k,j)] = E(i,j), taking the coefficient of E(y) on both sides
yields the 4-term formula in the structure. This is verified by direct algebra:
each of the 4 bracket cases (IA, IB, IIA, IIB) corresponds to one of the
4 RHS terms in `half_deriv_cond`.

### `omega` limitations with variable `n`

`omega` cannot prove `x.i ≠ n` when `n` is a variable (only concrete naturals).
Fix: use `Nat.lt_of_lt_of_le` followed by `omega`:

```lean4
have hx_i_lt_n : x.i < n := lt_of_lt_of_le x.hlt x.hle
have hx_i_ne_n : x.i ≠ n := by omega
```
