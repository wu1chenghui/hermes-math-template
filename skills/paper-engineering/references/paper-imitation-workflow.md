# Paper Imitation Workflow — Reference-Paper-Driven Rewriting

When rewriting a paper to match the style of specific reference papers,
follow this workflow developed during the HalfDer(N_n) rewrite project.

## Phase 1: Forensic Analysis

Before writing a single word, extract the reference papers' DNA:

1. **Structure analysis**: Section count, subsection boundaries, what goes where
2. **Notation audit**: Every symbol checked against reference papers for conflicts
3. **Sentence-level patterns**: Openers, transitions, closers — by frequency count
4. **Voice fingerprint**: Hedging count, "we" vs passive ratio, evaluation words
5. **Decision reconstruction**: What did the author choose NOT to write?
6. **Negative space analysis**: 10 categories of content the author omits
7. **Sentence metrics**: 8 quantifiable indicators (math%, subject variety, breathing pattern)

## Phase 2: Writing Exercises

Write individual sections in the target voice:

1. **Introduction** — match the reference paper's paragraph structure exactly
2. **Construction section** (§2) — match the (A)-(E) construction format
3. **Short lemma proof** — match the 6-step computation template
4. **Main proof** — match the cognitive breathing pattern (deep→flat alternation)

## Phase 3: Metric Audit

Run the 8 sentence metrics against each exercise. Fix violations iteratively.
